# UI UX Pro Max

Use the **UI UX Pro Max** design intelligence skill for this request.

1. Apply the design system workflow: infer product/industry → choose pattern, style, colors, typography, effects → list anti-patterns to avoid.
2. Output a short design system summary (TARGET, PATTERN, STYLE, COLORS, TYPOGRAPHY, EFFECTS, AVOID).
3. Implement the UI (or the part the user asked for) following that system.
4. Run the pre-delivery checklist before finishing (no emoji icons, cursor-pointer on clickables, hover/transition, contrast, focus states, reduced-motion, responsive).

If the user did not specify a product or page, ask for a one-line description (e.g. "SaaS dashboard for HR analytics") or infer from the codebase, then proceed.
