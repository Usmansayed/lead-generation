# Workflow: UI UX Pro Max

**Trigger:** User asks to build/design UI (landing page, dashboard, app, design system, colors, typography).

**Steps:**

1. **Activate** the UI UX Pro Max skill (auto-applied for UI/UX requests, or run the `/ui-ux-pro-max` command).
2. **Infer** product/industry from the request or codebase.
3. **Generate design system:** pattern → style → colors → typography → effects → anti-patterns.
4. **Output** a short design system block (TARGET, PATTERN, STYLE, COLORS, TYPOGRAPHY, EFFECTS, AVOID).
5. **Implement** the UI using the chosen stack (default HTML + Tailwind unless specified).
6. **Check** pre-delivery checklist (icons, pointer, hover, contrast, focus, reduced-motion, responsive).

**Skill:** `.cursor/skills/ui-ux-pro-max/SKILL.md`  
**Command:** `/ui-ux-pro-max` (`.cursor/commands/ui-ux-pro-max.md`)
